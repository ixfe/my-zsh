### 安装 packer

```

```

### 安装 coc

```
安装nvim
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.3/install.sh | bash
设置taobao源
npm config set registry=https://registry.npm.taobao.org
安装node
sudo npm i -g neovim yarn
```
